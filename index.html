ข้อมูลระบบ

เว็บไซต์ระบบบริหารจัดการส่วนกลางหมู่บ้าน เจซีการ์เดนวิลล์ (JC Garden Ville) ที่ช่วยให้กรรมการหมู่บ้านและนิติบุคคลสามารถจัดการข้อมูล ตรวจสอบรายละเอียดการแจ้งเรื่อง การเงิน และระบบส่วนกลางต่างๆ ได้อย่างสะดวก โปร่งใส และรวดเร็วในระบบเดียว

โครงสร้างหลักของเว็บไซต์หมู่บ้าน

เหตุผลที่เลือกทำโปรเจกต์นี้

  • แก้ปัญหาความยุ่งยาก — เปลี่ยนระบบบริหารงานส่วนกลางของกรรมการหมู่บ้านแบบเดิม ให้มาทำงานบนระบบดิจิทัลที่รวบรวมข้อมูลไว้ในที่เดียว
  • เพิ่มความโปร่งใส — ทำให้การจัดการข้อมูล การตรวจสอบรายละเอียดต่างๆ ของหมู่บ้านทำได้ง่าย และมีประสิทธิภาพมากขึ้น

หน้าที่หลักของฉันในกลุ่ม

รับผิดชอบเขียนโค้ดพัฒนาระบบหลังบ้าน (Backend) ด้วยภาษา PHP และการจัดการระบบฐานข้อมูลผ่าน SQL บนโปรแกรม VS Code
